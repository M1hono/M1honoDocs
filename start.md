# 🚀 快速启动指南

## 1. 安装依赖
```bash
npm install
```

## 2. 启动开发服务器
```bash
npm run dev
```

## 3. 在浏览器中访问
```
http://localhost:3000
```

## 4. 部署到GitHub Pages

### 方法1: 自动部署 (推荐)
1. 将代码推送到GitHub仓库的 `main` 分支
2. GitHub Actions会自动构建和部署
3. 在仓库设置中启用GitHub Pages，选择 `gh-pages` 分支

### 方法2: 手动部署
```bash
npm run deploy
```
